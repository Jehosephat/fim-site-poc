## Notes

- All text should be written in a casual, authentic tone matching the artist's brand
- Use emojis strategically to match the artist's style
- Keep descriptions concise but engaging
- Ensure all URLs are complete and correct
- Color names should be Tailwind CSS color names (without the number, e.g., "emerald" not "emerald-400")
- Image filenames should match exactly what's in the assets folder
